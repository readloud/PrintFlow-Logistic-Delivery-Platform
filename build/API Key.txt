## 1. Midtrans (Pembayaran Non-COD)

Midtrans digunakan untuk memproses pembayaran non-COD seperti transfer bank, QRIS, dan e-wallet.

### Langkah-langkah:

2. **Klik "Register"** di pojok kanan atas
3. **Isi formulir pendaftaran**:

   - Nama lengkap
   - Nomor telepon

5. **Login ke Merchant Administration Portal (MAP)**

### Mendapatkan Access Keys:

```bash

1. Buka Settings → Access Keys
2. Pilih Environment: "Sandbox" (untuk testing) atau "Production" (live)
3. Copy:
   - Server Key   → untuk backend (rahasiakan!)
   - Client Key   → untuk frontend/mobile
```

**Perbedaan Server Key vs Client Key**:
| Key | Kegunaan | Kerahasiaan |
|-----|----------|-------------|
| **Server Key** | Verifikasi transaksi, webhook, request API server-side | **WAJIB RAHASIA**, simpan di .env |
| **Client Key** | Inisialisasi Snap UI di frontend | Bisa publik, aman untuk client |

### Konfigurasi di `.env`:
```env
MIDTRANS_SERVER_KEY=SB-Mid-server-xxxxx
MIDTRANS_CLIENT_KEY=SB-Mid-client-xxxxx
MIDTRANS_IS_PRODUCTION=false   # true untuk production
```

> **Catatan**: Gunakan **Sandbox mode** untuk development, **Production** untuk live setelah semua testing selesai.

## ⚠️ Security Best Practices

1. **Jangan commit API key ke GitHub** - selalu gunakan `.env` dan `.gitignore`
2. **Restrict Google Maps API key** - set domain/IP whitelist 
3. **Gunakan environment yang berbeda** - Sandbox untuk development, Production untuk live
4. **Rotasi key secara berkala** - ganti API key setiap 3-6 bulan
5. **Monitor usage dashboard** - pantau tiap service untuk deteksi anomaly

# 🔐 AWS API Gateway & JWT Complete Integration Guide

## 📋 Overview

Sistem PRINTFLOW menggunakan **Amazon API Gateway** sebagai entry point semua request, dengan **JWT (JSON Web Token)** sebagai mekanisme autentikasi. JWT di-validate oleh **Lambda Authorizer** sebelum request diteruskan ke backend .

## 2. Lambda Authorizer (JWT Validator)

Lambda Authorizer memverifikasi JWT dari setiap request sebelum diteruskan ke backend.

### Lambda Authorizer Code (Node.js):

```javascript
// lambda/authorizer/index.js
const jwt = require('jsonwebtoken');
const jwksClient = require('jwks-rsa');

// Configuration - dari environment variables
const COGNITO_USER_POOL_ID = process.env.COGNITO_USER_POOL_ID;
const COGNITO_REGION = process.env.COGNITO_REGION || 'ap-southeast-1';
const ISSUER = `https://cognito-idp.${COGNITO_REGION}.amazonaws.com/${COGNITO_USER_POOL_ID}`;

// JWKS Client untuk fetch public keys dari Cognito
const client = jwksClient({
  jwksUri: `${ISSUER}/.well-known/jwks.json`,
  cache: true,
  cacheMaxAge: 86400000, // 24 hours
  rateLimit: true
});

// Helper function untuk get signing key
function getSigningKey(kid, callback) {
  client.getSigningKey(kid, (err, key) => {
    if (err) {
      console.error('Error getting signing key:', err);
      return callback(err);
    }
    const signingKey = key.getPublicKey();
    callback(null, signingKey);
  });
}

// Main authorizer handler
exports.handler = async (event) => {
  console.log('Authorizer event:', JSON.stringify(event, null, 2));
  
  let token = null;
  
  // Extract token from Authorization header
  if (event.authorizationToken) {
    const parts = event.authorizationToken.split(' ');
    if (parts.length === 2 && parts[0] === 'Bearer') {
      token = parts[1];
    }
  }
  
  // Alternative: extract from headers or query string
  if (!token && event.headers) {
    const authHeader = event.headers.Authorization || event.headers.authorization;
    if (authHeader) {
      const parts = authHeader.split(' ');
      if (parts.length === 2 && parts[0] === 'Bearer') {
        token = parts[1];
      }
    }
  }
  
  if (!token) {
    console.log('No token provided');
    return generatePolicy('user', 'Deny', event.methodArn, {
      message: 'No authorization token provided'
    });
  }
  
  try {
    // Decode token tanpa verifikasi untuk mendapatkan kid dan claims
    const decodedToken = jwt.decode(token, { complete: true });
    
    if (!decodedToken || !decodedToken.header || !decodedToken.header.kid) {
      console.log('Invalid token format');
      return generatePolicy('user', 'Deny', event.methodArn, {
        message: 'Invalid token format'
      });
    }
    
    // Get signing key dan verify token
    const signingKey = await new Promise((resolve, reject) => {
      getSigningKey(decodedToken.header.kid, (err, key) => {
        if (err) reject(err);
        else resolve(key);
      });
    });
    
    // Verify JWT
    const verified = jwt.verify(token, signingKey, {
      issuer: ISSUER,
      audience: process.env.COGNITO_APP_CLIENT_ID,
      algorithms: ['RS256']
    });
    
    console.log('Token verified successfully:', verified);
    
    // Extract user info from token claims
    const userId = verified.sub;
    const userEmail = verified.email;
    const userRole = verified['custom:role'] || 'USER';
    const userPhone = verified.phone_number;
    
    // Check if token is blacklisted (revoked) - optional Redis check
    const isBlacklisted = await checkBlacklist(token);
    if (isBlacklisted) {
      console.log('Token is blacklisted');
      return generatePolicy(userId, 'Deny', event.methodArn, {
        message: 'Token has been revoked'
      });
    }
    
    // Check if user has permission for the requested resource
    const hasPermission = await checkPermissions(
      userRole,
      event.methodArn,
      verified.scope
    );
    
    if (!hasPermission) {
      console.log('User does not have permission:', userRole, event.methodArn);
      return generatePolicy(userId, 'Deny', event.methodArn, {
        message: 'Insufficient permissions'
      });
    }
    
    // Generate Allow policy
    return generatePolicy(userId, 'Allow', event.methodArn, {
      sub: userId,
      email: userEmail,
      role: userRole,
      phone: userPhone,
      scope: verified.scope || '',
      'custom:claims': JSON.stringify(verified)
    });
    
  } catch (error) {
    console.error('Token verification failed:', error);
    
    if (error.name === 'TokenExpiredError') {
      return generatePolicy('user', 'Deny', event.methodArn, {
        message: 'Token has expired'
      });
    }
    
    return generatePolicy('user', 'Deny', event.methodArn, {
      message: 'Invalid token',
      error: error.message
    });
  }
};

// Helper: Generate IAM Policy untuk API Gateway
function generatePolicy(principalId, effect, resource, context = {}) {
  const policyDocument = {
    Version: '2012-10-17',
    Statement: [
      {
        Action: 'execute-api:Invoke',
        Effect: effect,
        Resource: resource
      }
    ]
  };
  
  // Untuk multiple resources (wildcard support)
  if (resource.includes('*')) {
    policyDocument.Statement[0].Resource = resource;
  }
  
  console.log('Generated policy:', JSON.stringify({
    principalId,
    effect,
    resource,
    context
  }));

  return {
    principalId: principalId,
    policyDocument: policyDocument,
    context: context
  };
}

// Helper: Check user permissions berdasarkan role dan scope
async function checkPermissions(userRole, methodArn, scope) {
  // Parse methodArn untuk mendapatkan resource path dan method
  // Format: arn:aws:execute-api:region:account-id:api-id/stage-name/HTTP_METHOD/resource-path
  
  const parts = methodArn.split('/');
  const method = parts[3]; // GET, POST, PUT, DELETE
  const resourcePath = '/' + parts.slice(4).join('/');
  
  console.log(`Checking permission for ${userRole} to ${method} ${resourcePath}`);
  
  // Role-based permission mapping
  const rolePermissions = {
    'ADMIN': {
      paths: ['*'],
      methods: ['*']
    },
    'PARTNER': {
      paths: ['/api/partner/*', '/api/orders/*'],
      methods: ['GET', 'POST', 'PUT']
    },
    'DRIVER': {
      paths: ['/api/driver/*', '/api/deliveries/*'],
      methods: ['GET', 'POST', 'PUT']
    },
    'USER': {
      paths: ['/api/user/*', '/api/orders/my-orders', '/api/payment/*'],
      methods: ['GET', 'POST']
    }
  };
  
  const permissions = rolePermissions[userRole] || rolePermissions['USER'];
  
  // Check if resource path matches
  let pathMatch = false;
  for (const allowedPath of permissions.paths) {
    if (allowedPath === '*') {
      pathMatch = true;
      break;
    }
    if (resourcePath === allowedPath || 
        (allowedPath.endsWith('/*') && resourcePath.startsWith(allowedPath.slice(0, -1)))) {
      pathMatch = true;
      break;
    }
    if (resourcePath.startsWith(allowedPath)) {
      pathMatch = true;
      break;
    }
  }
  
  // Check if method allowed
  const methodMatch = permissions.methods.includes('*') || 
                      permissions.methods.includes(method);
  
  // Check OAuth scopes (if present)
  let scopeMatch = true;
  if (scope) {
    const requiredScopes = getRequiredScopes(resourcePath, method);
    const userScopes = scope.split(' ');
    scopeMatch = requiredScopes.every(s => userScopes.includes(s));
  }
  
  return pathMatch && methodMatch && scopeMatch;
}

// Helper: Get required scopes for a given endpoint
function getRequiredScopes(resourcePath, method) {
  const scopeMap = {
    'GET /api/orders': ['read:orders'],
    'POST /api/orders': ['write:orders'],
    'PUT /api/orders': ['write:orders'],
    'DELETE /api/orders': ['delete:orders'],
    'GET /api/payments': ['read:payments'],
    'POST /api/payments': ['write:payments'],
    'GET /api/admin': ['admin:access'],
    'POST /api/admin': ['admin:access']
  };
  
  const key = `${method} ${resourcePath}`;
  return scopeMap[key] || [];
}
```

Backend Anda juga bisa memverifikasi JWT secara langsung sebagai lapisan keamanan kedua.

### JWT Middleware untuk Backend:

```javascript
// backend/src/middleware/jwt.middleware.js
const jwt = require('jsonwebtoken');
const jwksClient = require('jwks-rsa');

const COGNITO_REGION = process.env.COGNITO_REGION || 'ap-southeast-1';
const COGNITO_USER_POOL_ID = process.env.COGNITO_USER_POOL_ID;
const ISSUER = `https://cognito-idp.${COGNITO_REGION}.amazonaws.com/${COGNITO_USER_POOL_ID}`;
const AUDIENCE = process.env.COGNITO_APP_CLIENT_ID;

// JWKS Client
const client = jwksClient({
  jwksUri: `${ISSUER}/.well-known/jwks.json`,
  cache: true,
  cacheMaxAge: 86400000 // 24 hours
});

// Get signing key
function getSigningKey(kid, callback) {
  client.getSigningKey(kid, (err, key) => {
    if (err) return callback(err);
    const signingKey = key.getPublicKey();
    callback(null, signingKey);
  });
}

// Async function untuk verify token
async function verifyToken(token) {
  return new Promise((resolve, reject) => {
    const decoded = jwt.decode(token, { complete: true });
    
    if (!decoded || !decoded.header || !decoded.header.kid) {
      return reject(new Error('Invalid token format'));
    }
    
    getSigningKey(decoded.header.kid, (err, key) => {
      if (err) return reject(err);
      
      jwt.verify(token, key, {
        issuer: ISSUER,
        audience: AUDIENCE,
        algorithms: ['RS256']
      }, (verifyErr, verified) => {
        if (verifyErr) return reject(verifyErr);
        resolve(verified);
      });
    });
  });
}

// Express middleware
async function jwtAuthMiddleware(req, res, next) {
  try {
    // Get token from header
    const authHeader = req.headers.authorization;
    if (!authHeader || !authHeader.startsWith('Bearer ')) {
      return res.status(401).json({ error: 'No token provided' });
    }
    
    const token = authHeader.split(' ')[1];
    
    // Verify token
    const decoded = await verifyToken(token);
    
    // Attach user info to request
    req.user = {
      id: decoded.sub,
      email: decoded.email,
      role: decoded['custom:role'] || 'USER',
      phone: decoded.phone_number,
      claims: decoded
    };
    
    next();
  } catch (error) {
    console.error('JWT verification error:', error);
    
    if (error.name === 'TokenExpiredError') {
      return res.status(401).json({ error: 'Token expired' });
    }
    if (error.name === 'JsonWebTokenError') {
      return res.status(401).json({ error: 'Invalid token' });
    }
    
    res.status(401).json({ error: 'Authentication failed' });
  }
}

// Role-based middleware
function requireRole(...allowedRoles) {
  return (req, res, next) => {
    if (!req.user) {
      return res.status(401).json({ error: 'Unauthorized' });
    }
    
    if (allowedRoles.includes(req.user.role)) {
      next();
    } else {
      res.status(403).json({ 
        error: 'Forbidden',
        requiredRoles: allowedRoles,
        userRole: req.user.role
      });
    }
  };
}

// Scope-based middleware
function requireScope(requiredScope) {
  return (req, res, next) => {
    const userScopes = req.user.claims?.scope?.split(' ') || [];
    
    if (userScopes.includes(requiredScope) || userScopes.includes('*:*')) {
      next();
    } else {
      res.status(403).json({ 
        error: 'Insufficient scope',
        requiredScope,
        userScopes
      });
    }
  };
}

module.exports = {
  jwtAuthMiddleware,
  requireRole,
  requireScope,
  verifyToken
};
```

```dart
// lib/services/auth_service.dart
import 'package:amazon_cognito_identity_dart_2/cognito.dart';
import 'package:shared_preferences/shared_preferences.dart';

class AuthService {
  static final AuthService _instance = AuthService._internal();
  factory AuthService() => _instance;
  AuthService._internal();
  
  late CognitoUserPool _userPool;
  CognitoUser? _currentUser;
  
  // Initialize Cognito
  void init() {
    _userPool = CognitoUserPool(
      'ap-southeast-1_XXXXXXXXX', // User Pool ID
      'xxxxxxxxxxxxxxx',          // App Client ID
      'your-client-secret',       // Client Secret (if any)
    );
  }
  
  // Sign up
  Future<void> signUp(String email, String password, Map<String, String> attributes) async {
    final user = CognitoUser(email, _userPool);
    final signUpResult = await user.signUp(password, userAttributes: attributes);
    
    if (!signUpResult.userConfirmed) {
      // Need confirmation
      await _sendConfirmationCode(email);
    }
  }
  
  // Sign in
  Future<Map<String, String>> signIn(String username, String password) async {
    final user = CognitoUser(username, _userPool);
    _currentUser = user;
    
    final authDetails = AuthenticationDetails(username: username, password: password);
    final session = await user.authenticateUser(authDetails);
    
    // Get tokens
    final accessToken = session.getAccessToken().getJwtToken();
    final idToken = session.getIdToken().getJwtToken();
    final refreshToken = session.getRefreshToken().getToken();
    
    // Store tokens securely
    final prefs = await SharedPreferences.getInstance();
    await prefs.setString('access_token', accessToken);
    await prefs.setString('id_token', idToken);
    await prefs.setString('refresh_token', refreshToken);
    
    // Decode ID Token untuk mendapatkan user info
    final userInfo = _decodeJWT(idToken);
    
    return {
      'accessToken': accessToken,
      'idToken': idToken,
      'userId': userInfo['sub'],
      'email': userInfo['email'],
      'role': userInfo['custom:role'] ?? 'USER'
    };
  }
  
  // Refresh token
  Future<String> refreshSession() async {
    final prefs = await SharedPreferences.getInstance();
    final refreshToken = prefs.getString('refresh_token');
    
    if (refreshToken == null) throw Exception('No refresh token');
    
    final user = CognitoUser('', _userPool);
    final session = await user.refreshSession(refreshToken);
    
    final newAccessToken = session.getAccessToken().getJwtToken();
    await prefs.setString('access_token', newAccessToken);
    
    return newAccessToken;
  }
  
  // Sign out
  Future<void> signOut() async {
    await _currentUser?.signOut();
    final prefs = await SharedPreferences.getInstance();
    await prefs.remove('access_token');
    await prefs.remove('id_token');
    await prefs.remove('refresh_token');
  }
  
  // Get current access token
  Future<String?> getAccessToken() async {
    final prefs = await SharedPreferences.getInstance();
    return prefs.getString('access_token');
  }
  
  // Helper: decode JWT
  Map<String, dynamic> _decodeJWT(String token) {
    final parts = token.split('.');
    if (parts.length != 3) throw Exception('Invalid token');
    
    final payload = _base64UrlDecode(parts[1]);
    return json.decode(payload);
  }
  
  String _base64UrlDecode(String input) {
    String normalized = input.replaceAll('-', '+').replaceAll('_', '/');
    return utf8.decode(base64.decode(normalized));
  }
}
```

```dart
      baseUrl: 'https://api.printflow.com',
      connectTimeout: Duration(seconds: 30),
      receiveTimeout: Duration(seconds: 30),
      headers: {
        'Content-Type': 'application/json',
      },
    ));
    
    // Add logging interceptor
    _dio.interceptors.add(PrettyDioLogger(
      requestHeader: true,
      requestBody: true,
      responseBody: true,
    ));
    
    // Add JWT interceptor
    _dio.interceptors.add(InterceptorsWrapper(
      onRequest: (options, handler) async {
        final token = await AuthService().getAccessToken();
        if (token != null) {
          options.headers['Authorization'] = 'Bearer $token';
        }
        return handler.next(options);
      },
      onError: (DioError error, handler) async {
        // Handle 401 - Token expired
        if (error.response?.statusCode == 401) {
          try {
            // Try to refresh token
            final newToken = await AuthService().refreshSession();
            // Retry original request with new token
            final options = error.requestOptions;
            options.headers['Authorization'] = 'Bearer $newToken';
            final response = await _dio.fetch(options);
            return handler.resolve(response);
          } catch (refreshError) {
            // Refresh failed, redirect to login
            // Emit event to show login screen
            return handler.reject(error);
          }
        }
        return handler.next(error);
      },
    ));
  }
  
  Future<Response> get(String path, {Map<String, dynamic>? queryParams}) async {
    return await _dio.get(path, queryParameters: queryParams);
  }
  
  Future<Response> post(String path, {dynamic data}) async {

  }
```

## 6. JWT Logout & Blacklist Management

Untuk logout yang aman dan **single logout across devices**, kita perlu maintain token blacklist.

### Redis-based Token Blacklist:

```javascript
// backend/src/services/tokenBlacklist.service.js
const redis = require('redis');

class TokenBlacklistService {
  constructor() {
    this.client = redis.createClient({
      url: process.env.REDIS_URL
    });
    this.client.connect();
  }
  
  // Blacklist a token
  async blacklistToken(token, expiresInSeconds) {
    // Store token in Redis with TTL = remaining token lifetime
    await this.client.setEx(`blacklist:${token}`, expiresInSeconds, '1');
    console.log(`Token blacklisted: ${token.substring(0, 20)}...`);
  }
  
  // Check if token is blacklisted
  async isTokenBlacklisted(token) {
    const result = await this.client.get(`blacklist:${token}`);
    return result === '1';
  }
  
  // Blacklist all tokens for a user (logout from all devices)
  async blacklistUserTokens(userId) {
    // Generate a new token version for the user
    const newVersion = Date.now().toString();
    await this.client.set(`user:${userId}:token_version`, newVersion);
    return newVersion;
  }
  
  // Get user's current token version
  async getUserTokenVersion(userId) {
    const version = await this.client.get(`user:${userId}:token_version`);
    return version || '0';
  }
  
  // Check if token version matches user's current version
  async isTokenValidForUser(userId, tokenVersion) {
    const currentVersion = await this.getUserTokenVersion(userId);
    return tokenVersion === currentVersion;
  }
}

module.exports = new TokenBlacklistService();
```

### C. Middleware untuk RBAC di Backend

```javascript
// backend/src/middleware/rbac.middleware.js
const rolePermissions = {
  'ADMIN': {
    resources: ['*'],
    actions: ['*']
  },
  'PARTNER': {
    resources: ['/api/partner/*', '/api/orders/*'],
    actions: ['GET', 'POST', 'PUT']
  },
  'DRIVER': {
    resources: ['/api/driver/*', '/api/deliveries/*'],
    actions: ['GET', 'POST', 'PUT']
  },
  'USER': {
    resources: ['/api/user/*', '/api/orders/my-orders'],
    actions: ['GET', 'POST']
  }
};

function requireRole(...allowedRoles) {
  return (req, res, next) => {
    const userRole = req.user?.role || 'USER';
    
    if (allowedRoles.includes(userRole)) {
      return next();
    }
    
    res.status(403).json({
      error: 'Forbidden',
      message: `Role ${userRole} tidak memiliki akses. Diperlukan: ${allowedRoles.join(', ')}`
    });
  };
}

function requirePermission(resource, action) {
  return (req, res, next) => {
    const userRole = req.user?.role || 'USER';
    const permissions = rolePermissions[userRole];
    
    if (!permissions) {
      return res.status(403).json({ error: 'No permissions defined for role' });
    }
    
    const hasResourceAccess = permissions.resources.some(r => {
      if (r === '*') return true;
      if (r.endsWith('/*')) return resource.startsWith(r.slice(0, -1));
      return resource === r;
    });
    
    const hasActionAccess = permissions.actions.includes('*') || 
                            permissions.actions.includes(action);
    
    if (hasResourceAccess && hasActionAccess) {
      return next();
    }
    
    res.status(403).json({
      error: 'Forbidden',
      message: `Insufficient permission for ${action} ${resource}`
    });
  };
}

module.exports = { requireRole, requirePermission };
```

### D. Flutter Implementation

```dart
// lib/services/social_auth_service.dart
import 'package:amazon_cognito_identity_dart_2/cognito.dart';
import 'package:google_sign_in/google_sign_in.dart';
import 'package:facebook_auth/facebook_auth.dart';
import 'package:sign_in_with_apple/sign_in_with_apple.dart';

class SocialAuthService {
  final CognitoUserPool _userPool;
  
  SocialAuthService(this._userPool);
  
  // Google Sign In
  Future<CognitoUserSession> signInWithGoogle() async {
    // 1. Login with Google
    final GoogleSignIn googleSignIn = GoogleSignIn(
      scopes: ['email', 'profile'],
    );
    
    final GoogleSignInAccount? googleUser = await googleSignIn.signIn();
    final GoogleSignInAuthentication googleAuth = 
        await googleUser!.authentication;
    
    // 2. Exchange with Cognito
    final cognitoUser = CognitoUser(googleUser.email, _userPool);
    
    final authDetails = AuthenticationDetails.fromGoogle(
      idToken: googleAuth.idToken!,
      accessToken: googleAuth.accessToken,
    );
    
    final session = await cognitoUser.authenticateUser(authDetails);
    return session;
  }
  
  // Facebook Sign In
  Future<CognitoUserSession> signInWithFacebook() async {
    // 1. Login with Facebook
    final LoginResult result = await FacebookAuth.instance.login();
    
    if (result.status == LoginStatus.ok) {
      final userData = await FacebookAuth.instance.getUserData();
      
      // 2. Exchange with Cognito
      final cognitoUser = CognitoUser(userData['email'], _userPool);
      
      final authDetails = AuthenticationDetails.fromFacebook(
        accessToken: result.accessToken!.token,
      );
      
      final session = await cognitoUser.authenticateUser(authDetails);
      return session;
    }
    
    throw Exception('Facebook login failed');
  }
  
  // Sign In with Apple
  Future<CognitoUserSession> signInWithApple() async {
    // 1. Login with Apple
    final credential = await SignInWithApple.getAppleIDCredential(
      scopes: [
        AppleIDAuthorizationScopes.email,
        AppleIDAuthorizationScopes.fullName,
      ],
      webAuthenticationOptions: WebAuthenticationOptions(
        clientId: 'com.printflow.app',
        redirectUri: Uri.parse('https://printflow.com/callback'),
      ),
    );
    
    // 2. Exchange with Cognito
    final cognitoUser = CognitoUser(credential.email!, _userPool);
    
    final authDetails = AuthenticationDetails.fromApple(
      identityToken: credential.identityToken,
      authorizationCode: credential.authorizationCode,
    );
    
    final session = await cognitoUser.authenticateUser(authDetails);
    return session;
  }
}
```

### D. Flutter MFA Implementation

```dart
// lib/services/mfa_service.dart
class MFAService {
  final CognitoUserPool _userPool;
  
  MFAService(this._userPool);
  
  // Setup TOTP (Google Authenticator)
  Future<String> setupTOTP(String username) async {
    final user = CognitoUser(username, _userPool);
    final associateResult = await user.associateSoftwareToken();
    
    // QR Code URI untuk ditampilkan ke user
    // otpauth://totp/PrintFlow:username?secret={secret}&issuer=PrintFlow
    final qrCodeUri = associateResult.getQRCodeURI();
    
    return qrCodeUri;
  }
  
  // Verify TOTP code
  Future<void> verifyTOTP(String username, String code) async {
    final user = CognitoUser(username, _userPool);
    await user.verifySoftwareToken(code);
    
    // Set MFA preference setelah verified
    await user.setUserMfaPreference(
      smsEnabled: false,
      softwareTokenEnabled: true,
    );
  }
  
  // Verify SMS MFA code
  Future<void> verifySMS(String username, String code) async {
    final user = CognitoUser(username, _userPool);
    await user.respondToMfaChallenge(code, mfaType: 'SMS_MFA');
  }
  
  // Set MFA preference user
  Future<void> setMFAPreference(String username, {
    bool smsEnabled = false,
    bool totpEnabled = true,
  }) async {
    final user = CognitoUser(username, _userPool);
    await user.setUserMfaPreference(
      smsEnabled: smsEnabled,
      softwareTokenEnabled: totpEnabled,
    );
  }
}
```

### E. Adaptive Authentication (Risk-Based MFA)

Authsignal dapat diintegrasikan dengan Cognito untuk adaptive MFA berdasarkan tingkat risiko :

```javascript
// Dengan Authsignal, Anda bisa implementasi:
// - Low risk (IP dikenal, device familiar) → No MFA
// - Medium risk (lokasi baru) → SMS OTP
// - High risk (anomaly detection) → TOTP + Email OTP

const authsignal = new AuthsignalClient({
  tenantId: process.env.AUTHSIGNAL_TENANT_ID,
  apiSecretKey: process.env.AUTHSIGNAL_API_KEY
});

async function adaptiveAuth(userId, action, context) {
  const { actionResult, challenges } = await authsignal.track({
    userId,
    action,
    deviceId: context.deviceId,
    ipAddress: context.ip,
    userAgent: context.userAgent
  });
  
  if (actionResult === "CHALLENGE_REQUIRED") {
    // Tampilkan challenge MFA sesuai risiko
    return { requiresMFA: true, challenges };
  }
  
  return { requiresMFA: false };
}
```

### G. Auth Service dengan Cognito

```dart
// lib/services/auth_service.dart
import 'package:amazon_cognito_identity_dart_2/cognito.dart';
import 'package:flutter_secure_storage/flutter_secure_storage.dart';
import 'package:shared_preferences/shared_preferences.dart';
import '../models/auth_result.dart';
import '../models/user.dart';

class AuthService {
  static const String _userPoolId = 'ap-southeast-1_XXXXXXXXX';
  static const String _clientId = 'xxxxxxxxxxxxxxx';
  static const String _region = 'ap-southeast-1';
  
  late final CognitoUserPool _userPool;
  final FlutterSecureStorage _secureStorage = const FlutterSecureStorage();
  
  AuthService() {
    _userPool = CognitoUserPool(_userPoolId, _clientId);
  }
  
  // Sign in with email and password
  Future<AuthResult> signInWithEmail(String email, String password, {bool rememberMe = false}) async {
    try {
      final user = CognitoUser(email, _userPool);
      final authDetails = AuthenticationDetails(
        username: email,
        password: password,
      );
      
      final session = await user.authenticateUser(authDetails);
      
      // Store tokens
      await _storeTokens(session, rememberMe);
      
      return AuthResult.success(
        user: User.fromCognito(user),
        session: session,
        needsMFA: false,
      );
    } catch (e) {
      return AuthResult.failure(_handleCognitoError(e));
    }
  }
  
  // Sign in with social provider
  Future<AuthResult> signInWithSocial(String provider, String idToken, String accessToken) async {
    try {
      final user = CognitoUser('', _userPool);
      
      late AuthenticationDetails authDetails;
      if (provider == 'Google') {
        authDetails = AuthenticationDetails.fromGoogle(
          idToken: idToken,
          accessToken: accessToken,
        );
      } else if (provider == 'Facebook') {
        authDetails = AuthenticationDetails.fromFacebook(
          accessToken: accessToken,
        );
      } else {
        throw Exception('Unsupported provider');
      }
      
      final session = await user.authenticateUser(authDetails);
      await _storeTokens(session, true);
      
      return AuthResult.success(
        user: User.fromCognito(user),
        session: session,
        needsMFA: false,
      );
    } catch (e) {
      return AuthResult.failure(_handleCognitoError(e));
    }
  }
  
  // Store tokens securely
  Future<void> _storeTokens(CognitoUserSession session, bool rememberMe) async {
    await _secureStorage.write(key: 'access_token', value: session.accessToken.jwtToken);
    await _secureStorage.write(key: 'id_token', value: session.idToken.jwtToken);
    await _secureStorage.write(key: 'refresh_token', value: session.refreshToken.token);
    
    if (rememberMe) {
      final prefs = await SharedPreferences.getInstance();
      await prefs.setBool('remember_me', true);
    }
  }
  
  // Get current session
  Future<CognitoUserSession?> getCurrentSession() async {
    final refreshToken = await _secureStorage.read(key: 'refresh_token');
    if (refreshToken == null) return null;
    
    try {
      final user = CognitoUser('', _userPool);
      final session = await user.refreshSession(refreshToken);
      return session;
    } catch (e) {
      return null;
    }
  }
  
  // Sign out
  Future<void> signOut() async {
    await _secureStorage.deleteAll();
    final prefs = await SharedPreferences.getInstance();
    await prefs.clear();
  }
  
  // Forgot password
  Future<bool> forgotPassword(String email) async {
    try {
      final user = CognitoUser(email, _userPool);
      await user.forgotPassword();
      return true;
    } catch (e) {
      return false;
    }
  }
  
  String _handleCognitoError(dynamic error) {
    if (error.toString().contains('UserNotFoundException')) {
      return 'User not found';
    }
    if (error.toString().contains('NotAuthorizedException')) {
      return 'Invalid email or password';
    }
    if (error.toString().contains('UserNotConfirmedException')) {
      return 'Please verify your email first';
    }
    if (error.toString().contains('CodeMismatchException')) {
      return 'Invalid verification code';
    }
    if (error.toString().contains('ExpiredCodeException')) {
      return 'Verification code expired';
    }
    if (error.toString().contains('LimitExceededException')) {
      return 'Too many attempts. Please try again later';
    }
    return 'Authentication failed: ${error.toString()}';
  }
}
```

```dart
    try {
      final GoogleSignInAccount? googleUser = await _googleSignIn.signIn();
      if (googleUser == null) {
        throw Exception('User cancelled Google sign in');
      }
      
      final GoogleSignInAuthentication googleAuth =

        'provider': 'Google',
        'idToken': googleAuth.idToken!,
        'accessToken': googleAuth.accessToken,
        'email': googleUser.email,
        'name': googleUser.displayName ?? '',
      };
    } catch (e) {
      throw Exception('Google sign in failed: $e');
    }
  }
  
  // Facebook Sign In
  Future<Map<String, String>> signInWithFacebook() async {
    try {
      final LoginResult result = await FacebookAuth.instance.login(
        permissions: ['public_profile', 'email'],
      );
      
      if (result.status == LoginStatus.ok) {
        final userData = await FacebookAuth.instance.getUserData(
          fields: 'email,name,picture',
        );
        
        return {
          'provider': 'Facebook',
          'accessToken': result.accessToken!.token,
          'email': userData['email'] as String,
          'name': userData['name'] as String,
        };
      } else if (result.status == LoginStatus.cancelled) {
        throw Exception('User cancelled Facebook sign in');
      } else {
        throw Exception('Facebook sign in failed');
      }
    } catch (e) {
      throw Exception('Facebook sign in failed: $e');
    }
  }
  
  // Sign In with Apple (iOS only)

    if (kIsWeb) {
      throw Exception('Sign in with Apple is not supported on web');
    }

          ),
        ),

        'authorizationCode': credential.authorizationCode,
        'email': credential.email ?? '',
        'name': '${credential.fullName?.givenName ?? ''} ${credential.fullName?.familyName ?? ''}'.trim(),
      };
    } catch (e) {
      throw Exception('Apple sign in failed: $e');
    }
  }
```

```javascript
// backend/src/services/authsignal.service.js
const axios = require('axios');

class AuthsignalService {
  constructor() {
    this.tenantId = process.env.AUTHSIGNAL_TENANT_ID;
    this.apiSecretKey = process.env.AUTHSIGNAL_API_KEY;
    this.baseUrl = 'https://api.authsignal.com/v1';
  }
  
  // Track user action and get risk assessment
  async trackAction(userId, action, context) {
    try {
      const response = await axios.post(
        `${this.baseUrl}/actions/${action}/track`,
        {
          userId,
          deviceId: context.deviceId,
          ipAddress: context.ipAddress,
          userAgent: context.userAgent,
          timestamp: new Date().toISOString(),
        },
        {
          headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${this.apiSecretKey}`,
            'X-Authsignal-Tenant-Id': this.tenantId,
          },
        }
      );
      
      return response.data;
    } catch (error) {
      console.error('Authsignal tracking error:', error);
      return { actionResult: 'ALLOW', challenges: [] };
    }
  }
  
  // Validate MFA challenge
  async validateChallenge(userId, challengeId, code) {
    try {
      const response = await axios.post(
        `${this.baseUrl}/actions/validate`,
        {
          userId,
          challengeId,
          code,
        },
        {
          headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${this.apiSecretKey}`,
            'X-Authsignal-Tenant-Id': this.tenantId,
          },
        }
      );
      
      return response.data;
    } catch (error) {
      console.error('Authsignal validation error:', error);
      return { isValid: false };
    }
  }
  
  // Get user's enrolled MFA methods
  async getUserMethods(userId) {
    try {
      const response = await axios.get(
        `${this.baseUrl}/users/${userId}/methods`,
        {
          headers: {
            'Authorization': `Bearer ${this.apiSecretKey}`,
            'X-Authsignal-Tenant-Id': this.tenantId,
          },
        }
      );
      
      return response.data;
    } catch (error) {
      console.error('Authsignal get methods error:', error);
      return { methods: [] };
    }
  }
  
  // Enroll new MFA method
  async enrollMethod(userId, methodType) {
    try {
      const response = await axios.post(
        `${this.baseUrl}/users/${userId}/methods`,
        {
          type: methodType, // 'SMS', 'TOTP', 'EMAIL', 'PASSKEY'
        },
        {
          headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${this.apiSecretKey}`,
            'X-Authsignal-Tenant-Id': this.tenantId,
          },
        }
      );
      
      return response.data;
```

```javascript
// backend/src/controllers/auth.controller.js
const authsignalService = require('../services/authsignal.service');
const { CognitoIdentityServiceProvider } = require('aws-sdk');

async function adaptiveLogin(req, res) {
  const { email, password, deviceId, ipAddress, userAgent } = req.body;
  
  try {
    // 1. First, authenticate with Cognito
    const cognito = new CognitoIdentityServiceProvider();
    const authResult = await cognito.adminInitiateAuth({
      UserPoolId: process.env.COGNITO_USER_POOL_ID,
      ClientId: process.env.COGNITO_CLIENT_ID,
      AuthFlow: 'ADMIN_NO_SRP_AUTH',
      AuthParameters: {
        USERNAME: email,
        PASSWORD: password,
      },
    }).promise();
    
    const userId = authResult.AuthenticationResult.AccessToken;
    
    // 2. Get user's IP and device info
    const context = {
      deviceId,
      ipAddress,
      userAgent,
    };
    
    // 3. Track login action with Authsignal for risk assessment
    const { actionResult, challenges, challengeId } = 
      await authsignalService.trackAction(userId, 'login', context);
    
    // 4. Based on risk assessment, determine if MFA is needed
    if (actionResult === 'CHALLENGE_REQUIRED') {
      return res.json({
        success: false,
        requiresMFA: true,
        challengeId,
        challenges, // ['SMS', 'TOTP', 'EMAIL']
        session: authResult.Session,
      });
    }
    
    // 5. Low risk - allow login
    return res.json({
      success: true,
      user: { id: userId, email },
      tokens: authResult.AuthenticationResult,
    });
    
  } catch (error) {
    console.error('Login error:', error);
    res.status(401).json({
      success: false,
      error: 'Authentication failed',
    });
  }
}

async function verifyMFA(req, res) {
  const { challengeId, code, method, session } = req.body;
  
  try {
    const isValid = await authsignalService.validateChallenge(
      req.user.id,
      challengeId,
      code
    );
    
    if (!isValid) {
      return res.status(401).json({
        success: false,
        error: 'Invalid verification code',
      });
    }
    
    // Continue with Cognito MFA verification
    const cognito = new CognitoIdentityServiceProvider();
    const result = await cognito.respondToAuthChallenge({
      ChallengeName: method === 'SMS' ? 'SMS_MFA' : 'SOFTWARE_TOKEN_MFA',
      ClientId: process.env.COGNITO_CLIENT_ID,
      Session: session,
      ChallengeResponses: {
        USERNAME: req.body.email,
        SMS_MFA_CODE: code,
        SOFTWARE_TOKEN_MFA_CODE: code,
      },
    }).promise();
    
    res.json({
      success: true,
      tokens: result.AuthenticationResult,
    });
    
  } catch (error) {
    res.status(401).json({
      success: false,
      error: 'MFA verification failed',
    });
  }
}

module.exports = { adaptiveLogin, verifyMFA };
```

## 🔧 3. TROUBLESHOOTING COGNITO ISSUES

### A. Common Errors & Solutions

| Error Code | Error Message | Penyebab | Solusi |
|------------|---------------|----------|---------|
| `UserNotFoundException` | User does not exist | Email tidak terdaftar | Validasi email sebelum login |
| `NotAuthorizedException` | Incorrect username or password | Password salah | Implementasi forgot password |
| `UserNotConfirmedException` | User is not confirmed | Email belum diverifikasi | Kirim ulang verification code |
| `CodeMismatchException` | Invalid verification code | Kode OTP salah | Auto-resend dengan countdown |
| `ExpiredCodeException` | Verification code expired | Kode kadaluarsa (10 menit) | Kirim ulang kode baru |
| `LimitExceededException` | Attempt limit exceeded | Terlalu banyak percobaan | Implementasi rate limiting + captcha |
| `InvalidParameterException` | Invalid parameter | Format input salah | Validasi frontend yang ketat |
| `ResourceNotFoundException` | User pool not found | Config salah | Check `USER_POOL_ID` dan `CLIENT_ID` |
| `AliasExistsException` | Email already exists | Email sudah terdaftar | Redirect ke login |

```dart
class AuthRetryService {

  static const baseDelay = Duration(seconds: 1);
  
  Future<T> withRetry<T>(Future<T> Function() operation) async {
    return retry(
      operation,
      maxAttempts: maxRetries,
      delayFactor: baseDelay,
      onRetry: (e) {
        print('Retrying due to: $e');
      },
      retryIf: (e) {
        // Retry only for specific errors

      },
    );
```

### D. Flutter Implementation untuk Passkey Login

```dart
// lib/services/passkey_service.dart
import 'package:cognito_passkey/cognito_passkey.dart';
import 'package:webview_flutter/webview_flutter.dart';

class PasskeyService {
  static const String cognitoDomain = 'https://login.printflow.com';
  static const String clientId = 'your-app-client-id';
  static const String redirectUri = 'com.printflow.app://callback';
  
  // Sign in with Passkey
  Future<Map<String, dynamic>> signInWithPasskey() async {
    try {
      // Check if passkey is available on device
      final isAvailable = await PasskeyAuth.isAvailable();
      if (!isAvailable) {
        throw Exception('Passkey not available on this device');
      }
      
      // Initiate passkey authentication via Cognito Hosted UI
      final authUrl = Uri.parse(
        '$cognitoDomain/oauth2/authorize?'
        'response_type=code&'
        'client_id=$clientId&'
        'redirect_uri=$redirectUri&'
        'identity_provider=WebAuthn'
      );
      
      // Use WebView for passkey authentication
      final result = await _authenticateWithWebView(authUrl);
      
      // Exchange authorization code for tokens
      final tokens = await _exchangeCodeForTokens(result['code']);
      
      return {
        'success': true,
        'accessToken': tokens['access_token'],
        'idToken': tokens['id_token'],
        'refreshToken': tokens['refresh_token'],
      };
    } catch (e) {
      print('Passkey authentication failed: $e');
      return {'success': false, 'error': e.toString()};
    }
  }
  
  // Register new passkey (after password login)
  Future<bool> registerPasskey(String accessToken) async {
    try {
      final isAvailable = await PasskeyAuth.isAvailable();
      if (!isAvailable) return false;
      
      // Initiate passkey registration via Cognito API
      final response = await _dio.post(
        '$cognitoDomain/webauthn/register',
        options: Options(
          headers: {'Authorization': 'Bearer $accessToken'},
        ),
      );
      
      final challenge = response.data['challenge'];
      final credential = await PasskeyAuth.register(challenge);
      
      // Verify registration with Cognito
      await _dio.post(
        '$cognitoDomain/webauthn/register/verify',
        data: credential,
        options: Options(
          headers: {'Authorization': 'Bearer $accessToken'},
        ),
      );
      
      return true;
    } catch (e) {
      print('Passkey registration failed: $e');
      return false;
    }
  }
  
  Future<Map<String, dynamic>> _exchangeCodeForTokens(String code) async {
    final response = await _dio.post(
      '$cognitoDomain/oauth2/token',
      data: {
        'grant_type': 'authorization_code',
        'client_id': clientId,
        'code': code,
        'redirect_uri': redirectUri,
      },
    );
    return response.data;
  }
}
```

## 3. LAMBDA TRIGGERS LENGKAP

Cognito menyediakan beberapa trigger poin dalam authentication flow :

### A. Jenis Lambda Triggers

| Trigger | Flow | Use Case |
|---------|------|----------|
| **Pre Sign-up** | Sign up | Validasi custom, reCAPTCHA, auto-confirm  |
| **Pre Authentication** | Sign in | Block user, custom validation, attempt counting  |
| **Post Authentication** | After sign in | Logging, sync to external system |
| **Post Confirmation** | After email verification | Auto-add to group, create profile  |
| **Custom Message** | Email/SMS sending | Custom email templates  |
| **Pre Token Generation** | Token creation | Add custom claims to JWT  |
| **Migrate User** | Import from legacy | Password migration |

```javascript
// lambda/cognito-triggers/index.js
const AWS = require('aws-sdk');
const cognito = new AWS.CognitoIdentityServiceProvider();

// Custom attributes untuk tracking 
const CUSTOM_ATTEMPTS_ATTR = 'custom:login_attempts';

exports.handler = async (event) => {
  console.log('Trigger:', event.triggerSource);
  console.log('Event:', JSON.stringify(event, null, 2));
  
  switch (event.triggerSource) {
    case 'PreSignUp_SignUp':
      return await preSignUpHandler(event);
    case 'PreSignUp_AdminCreateUser':
      return await preSignUpAdminHandler(event);
      
    case 'PreAuthentication_Authentication':
      return await preAuthenticationHandler(event);
      
    case 'PostAuthentication_Authentication':
      return await postAuthenticationHandler(event);
      
    case 'PostConfirmation_ConfirmSignUp':
      return await postConfirmationHandler(event);
      
    case 'PreTokenGeneration':
      return await preTokenGenerationHandler(event);
      
    case 'CustomMessage_SignUp':
    case 'CustomMessage_ForgotPassword':
    case 'CustomMessage_ResendCode':
    case 'CustomMessage_AdminCreateUser':
    case 'CustomMessage_Authentication':
      // Handled by separate custom-message lambda for clarity
      return event;
      
    default:
      console.log('Unhandled trigger source:', event.triggerSource);
      return event;
  }

// 1. PRE SIGN-UP: Validasi custom + reCAPTCHA 
async function preSignUpHandler(event) {
  const email = event.request.userAttributes.email;
  const customAttributes = event.request.userAttributes;
  
  // Validasi email domain (hanya untuk partner)
  if (customAttributes['custom:role'] === 'PARTNER') {
    const allowedDomains = ['printflow.com', 'partner.com'];
    const domain = email.split('@')[1];
    if (!allowedDomains.includes(domain)) {
      throw new Error('Email domain tidak diizinkan untuk role Partner');
    }
  }
  
  // Auto-confirm email dari domain tertentu
  if (email.endsWith('@printflow.com')) {
    event.response.autoConfirmUser = true;
    event.response.autoVerifyEmail = true;
  }
  
  // Tambahkan custom attributes default
  event.response.autoConfirmUser = false;
  
  return event;
}

// 2. PRE AUTHENTICATION: Block user & attempt counting 
async function preAuthenticationHandler(event) {
  const username = event.userName;
  const userPoolId = event.userPoolId;
  
  // Cek apakah user di-block
  const isUserEnabled = await isUserEnabled(userPoolId, username);
  if (!isUserEnabled) {
    throw new Error('Akun Anda telah diblokir. Silakan hubungi customer support.');
  }
  
  // Hitung login attempts 
  const attempts = await getLoginAttempts(userPoolId, username);
  const MAX_ATTEMPTS = 5;
  
  if (attempts >= MAX_ATTEMPTS) {
    // Block user setelah 5 attempt gagal 
    await disableUser(userPoolId, username);
    throw new Error('Terlalu banyak percobaan login. Akun Anda telah diblokir.');
  }
  
  // Update counter (akan di-reset jika login sukses di post-auth)
  await updateLoginAttempts(userPoolId, username, attempts + 1);
  
  return event;
}

// 3. POST AUTHENTICATION: Reset counter & logging 
async function postAuthenticationHandler(event) {
  const username = event.userName;
  const userPoolId = event.userPoolId;
  
  // Reset login attempts counter
  await resetLoginAttempts(userPoolId, username);
  
  // Log audit ke CloudWatch atau DynamoDB
  console.log(`User ${username} logged in successfully at ${new Date().toISOString()}`);
  
  // Opsional: Update last login timestamp
  await updateLastLogin(userPoolId, username);
  
  return event;
}

// 4. POST CONFIRMATION: Auto-add to group, create profile 
async function postConfirmationHandler(event) {
  const username = event.userName;
  const userPoolId = event.userPoolId;
  const userAttributes = event.request.userAttributes;
  const role = userAttributes['custom:role'] || 'USER';
  
  // Auto-add user ke Cognito group berdasarkan role
  await addUserToGroup(userPoolId, username, role);
  
  // Create user profile di database aplikasi
  await createUserProfile({
    cognitoId: event.request.userAttributes.sub,
    email: userAttributes.email,
    name: userAttributes.name,
    role: role,
    createdAt: new Date().toISOString()
  });
  
  console.log(`User ${username} confirmed and added to ${role} group`);
  
  return event;
}

// 5. PRE TOKEN GENERATION: Add custom claims to JWT 
async function preTokenGenerationHandler(event) {
  const userAttributes = event.request.userAttributes;
  const groups = event.request.groupConfiguration?.groupsToOverride || [];
  
  // Add custom claims ke access token
  event.response = {
    claimsOverrideDetails: {
      claimsToAddOrOverride: {
        'custom:role': userAttributes['custom:role'] || 'USER',
        'custom:user_id': userAttributes.sub,
        'scope': groups.map(g => `sc:${g.toLowerCase()}`).join(' '),
        'https://printflow.com/email_verified': userAttributes.email_verified,
      },
      claimsToSuppress: ['phone_number_verified'] // Hapus claim yang tidak perlu
    }
  };
  
  return event;
}

    UserPoolId: userPoolId,

    return false;
  }

// Get login attempts dari custom attribute 
async function getLoginAttempts(userPoolId, username) {
  const params = {
    UserPoolId: userPoolId,
    Username: username
  };
  
  try {
    const userData = await cognito.adminGetUser(params).promise();
    const attribute = userData.UserAttributes.find(
      (att) => att.Name === CUSTOM_ATTEMPTS_ATTR
    );
    return attribute ? parseInt(attribute.Value) : 0;
  } catch (error) {
    console.error('Error getting attempts:', error);
    return 0;
  }
}
```

## 1. RECAPTCHA DI PRE SIGN-UP LAMBDA

### A. Overview & Setup

reCAPTCHA mencegah bot sign-up dan brute force attacks. Ada dua versi:
- **reCAPTCHA v3** (invisible, berbasis skor) - Recommended untuk sign-up flow 
- **reCAPTCHA v2** ("I'm not robot" checkbox) - Alternatif

### E. Backend: Pre Sign-up Lambda dengan reCAPTCHA

```javascript
// lambda/pre-signup-recaptcha/index.js
const axios = require('axios');

// Konfigurasi
const RECAPTCHA_SECRET_KEY = process.env.RECAPTCHA_SECRET_KEY;
const RECAPTCHA_VERIFY_URL = 'https://www.google.com/recaptcha/api/siteverify';
const MINIMUM_SCORE = 0.5; // Minimum score untuk dianggap valid (0.0 - 1.0)

// reCAPTCHA v3 validation
async function validateRecaptchaV3(token, action) {
  try {
    const response = await axios.post(RECAPTCHA_VERIFY_URL, null, {
      params: {
        secret: RECAPTCHA_SECRET_KEY,
        response: token
      }
    });
    
    const data = response.data;
    
    if (!data.success) {
      console.error('reCAPTCHA validation failed:', data['error-codes']);
      return { valid: false, score: 0, error: data['error-codes'] };
    }
    
    // Check if action matches
    if (data.action !== action) {
      console.warn(`Action mismatch: expected ${action}, got ${data.action}`);
      return { valid: false, score: data.score, error: 'Action mismatch' };
    }
    
    // Check score threshold
    if (data.score < MINIMUM_SCORE) {
      console.warn(`Low score: ${data.score} (threshold: ${MINIMUM_SCORE})`);
      return { valid: false, score: data.score, error: 'Low score threshold' };
    }
    
    return { valid: true, score: data.score };
    
  } catch (error) {
    console.error('reCAPTCHA verify error:', error);
    return { valid: false, error: error.message };
  }
}

// reCAPTCHA v2 validation
async function validateRecaptchaV2(token) {
  try {
    const response = await axios.post(RECAPTCHA_VERIFY_URL, null, {
      params: {
        secret: RECAPTCHA_SECRET_KEY,
        response: token
      }
    });
    
    const data = response.data;
    
    if (!data.success) {
      console.error('reCAPTCHA v2 validation failed:', data['error-codes']);
      return { valid: false, error: data['error-codes'] };
    }
    
    return { valid: true };
    
  } catch (error) {
    console.error('reCAPTCHA v2 verify error:', error);
    return { valid: false, error: error.message };
  }
}

// Lambda handler untuk Pre Sign-up
exports.handler = async (event) => {
  console.log('PreSign-up trigger invoked:', JSON.stringify(event, null, 2));
  
  try {
    // Get reCAPTCHA token dari request (harus dikirim via user attributes)
    const recaptchaToken = event.request.userAttributes['custom:recaptcha_token'];
    const recaptchaVersion = event.request.userAttributes['custom:recaptcha_version'] || 'v3';
    const recaptchaAction = event.request.userAttributes['custom:recaptcha_action'] || 'signup';
    
    if (!recaptchaToken) {
      console.error('No reCAPTCHA token provided');
      throw new Error('reCAPTCHA verification required');
    }
    
    let validationResult;
    
    if (recaptchaVersion === 'v2') {
      validationResult = await validateRecaptchaV2(recaptchaToken);
    } else {
      validationResult = await validateRecaptchaV3(recaptchaToken, recaptchaAction);
    }
    
    if (!validationResult.valid) {
      console.error('reCAPTCHA validation failed:', validationResult.error);
      
      // Throw error yang akan diteruskan ke user
      throw new Error(`reCAPTCHA verification failed: ${validationResult.error || 'Invalid token'}`);
    }
    
    console.log('reCAPTCHA validated successfully. Score:', validationResult.score);
    
    // Remove temporary custom attributes
    delete event.request.userAttributes['custom:recaptcha_token'];
    delete event.request.userAttributes['custom:recaptcha_version'];
    delete event.request.userAttributes['custom:recaptcha_action'];
    
    // Optional: Add reCAPTCHA score to user attributes untuk monitoring
    if (validationResult.score) {
      event.response = {
        autoConfirmUser: false,
        autoVerifyEmail: false
      };
      
      // Store score as custom attribute (for analytics)
      event.request.userAttributes['custom:recaptcha_score'] = validationResult.score.toString();
    }
    
    return event;
    
  } catch (error) {
    console.error('PreSign-up error:', error);
    
    // Cognito akan mengirim error ini ke user
    throw new Error(`Pre-signup validation failed: ${error.message}`);
  }
};

// Lambda dengan rate limiting dan blacklist domain
const rateLimitManager = require('./rate-limit');

exports.handlerWithRateLimit = async (event) => {
  const email = event.request.userAttributes.email;
  const domain = email.split('@')[1];
  const ip = event.request.context?.sourceIp || 'unknown';
  
  // 1. Block disposable email domains
  const disposableDomains = [
    'tempmail.com', '10minutemail.com', 'guerrillamail.com',
    'mailinator.com', 'yopmail.com'
  ];
  
  if (disposableDomains.includes(domain)) {
    throw new Error('Email domain tidak diizinkan. Gunakan email permanen.');
  }
  
  // 2. Check rate limiting per IP
  const attempts = await rateLimitManager.getAttempts(ip);
  const MAX_ATTEMPTS_PER_HOUR = 5;
  
  if (attempts >= MAX_ATTEMPTS_PER_HOUR) {
    throw new Error('Terlalu banyak percobaan pendaftaran. Coba lagi nanti.');
  }
  
  await rateLimitManager.incrementAttempts(ip);
  
  // 3. Validate reCAPTCHA
  const recaptchaToken = event.request.userAttributes['custom:recaptcha_token'];
  const validation = await validateRecaptchaV3(recaptchaToken, 'signup');
  
  if (!validation.valid) {
    throw new Error('reCAPTCHA validation failed. Please try again.');
  }
  
  return event;
};
```

### F. Rate Limiting Helper (Redis)

```javascript
// lambda/pre-signup-recaptcha/rate-limit.js
const Redis = require('ioredis');

const redis = new Redis({
  host: process.env.REDIS_HOST || 'localhost',
  port: process.env.REDIS_PORT || 6379,
  password: process.env.REDIS_PASSWORD
});

class RateLimitManager {
  constructor() {
    this.windowMs = 60 * 60 * 1000; // 1 hour
  }
  
  async getAttempts(identifier) {
    const key = `signup_attempts:${identifier}`;
    const attempts = await redis.get(key);
    return attempts ? parseInt(attempts) : 0;
  }
  
  async incrementAttempts(identifier) {
    const key = `signup_attempts:${identifier}`;
    const attempts = await redis.incr(key);
    
    if (attempts === 1) {
      await redis.expire(key, this.windowMs / 1000);
    }
    
    return attempts;
  }
  
  async resetAttempts(identifier) {
    const key = `signup_attempts:${identifier}`;
    await redis.del(key);
  }
  
  // Block IP after many failed attempts
  async blockIP(ip, durationMinutes = 60) {
    const key = `blocked_ip:${ip}`;
    await redis.setex(key, durationMinutes * 60, '1');
  }
  
  async isIPBlocked(ip) {
    const key = `blocked_ip:${ip}`;
    const blocked = await redis.get(key);
    return blocked === '1';
  }
}

module.exports = new RateLimitManager();
```

```javascript
// backend/src/services/bulkEmail.service.js
const AWS = require('aws-sdk');
const { v4: uuidv4 } = require('uuid');
const path = require('path');
const fs = require('fs').promises;

class BulkEmailService {
  constructor() {
    this.ses = new AWS.SES({ region: 'ap-southeast-1' });
    this.s3 = new AWS.S3();
    this.sqs = new AWS.SQS();
    
    this.BULK_QUEUE_URL = process.env.BULK_EMAIL_QUEUE_URL;
    this.BUCKET_NAME = process.env.EMAIL_BUCKET_NAME;
  }
  
  // ============ CREATE BULK EMAIL CAMPAIGN ============
  async createCampaign(campaignData) {
    const campaignId = uuidv4();
    const campaign = {
      id: campaignId,
      name: campaignData.name,
      subject: campaignData.subject,
      templateName: campaignData.templateName,
      templateData: campaignData.templateData,
      recipientList: campaignData.recipientList, // Array of emails
      scheduledAt: campaignData.scheduledAt || new Date(),
      status: 'PENDING',
      createdAt: new Date(),
      totalRecipients: campaignData.recipientList.length,
      sentCount: 0,
      failedCount: 0
    };
    
    // Save to DynamoDB
    await this.saveCampaign(campaign);
    
    // If schedule is immediate or past, queue for sending
    if (new Date(campaign.scheduledAt) <= new Date()) {
      await this.queueCampaignForSending(campaignId);
    }
    
    return campaign;
  }
  
  // ============ QUEUE BULK EMAIL TO SQS ============
  async queueCampaignForSending(campaignId) {
    const params = {
      QueueUrl: this.BULK_QUEUE_URL,
      MessageBody: JSON.stringify({
        campaignId,
        action: 'START_CAMPAIGN'
      }),
      MessageGroupId: campaignId,
      MessageDeduplicationId: `${campaignId}-${Date.now()}`,
      DelaySeconds: 0
    };
    
    await this.sqs.sendMessage(params).promise();
    console.log(`Campaign ${campaignId} queued for sending`);
  }
  
  // ============ PROCESS BULK EMAIL BATCH ============
  async processBulkCampaign(campaignId, batchSize = 50) {
    const campaign = await this.getCampaign(campaignId);
    
    if (!campaign || campaign.status === 'COMPLETED' || campaign.status === 'CANCELLED') {
      return { error: 'Campaign not found or already completed' };
    }
    
    // Get unsent recipients
    const unsentRecipients = await this.getUnsentRecipients(campaignId);
    const batch = unsentRecipients.slice(0, batchSize);
    
    if (batch.length === 0) {
      await this.completeCampaign(campaignId);
      return { completed: true };
    }
    
    // Prepare bulk email request
    const destinations = batch.map(recipient => ({
      Destination: {
        ToAddresses: [recipient.email]
      },
      ReplacementTemplateData: JSON.stringify({
        name: recipient.name,
        email: recipient.email,
        unsubscribeLink: `https://printflow.com/unsubscribe?email=${encodeURIComponent(recipient.email)}&campaign=${campaignId}`,
        ...campaign.templateData
      })
    }));
    
    try {
      // Send bulk templated email (SES built-in bulk)
      const params = {
        Source: 'PrintFlow <newsletter@printflow.com>',
        Template: campaign.templateName,
        DefaultTemplateData: JSON.stringify({
          name: 'Valued Customer',
          unsubscribeLink: `https://printflow.com/unsubscribe?campaign=${campaignId}`
        }),
        Destinations: destinations,
        ConfigurationSetName: 'printflow-bulk-configuration'
      };
      
      const result = await this.ses.sendBulkTemplatedEmail(params).promise();
      
      // Update sent status untuk setiap recipient
      const updates = batch.map((recipient, index) => ({
        campaignId,
        email: recipient.email,
        status: result.Status[index] === 'Success' ? 'SENT' : 'FAILED',
        messageId: result.Status[index] === 'Success' ? result.MessageId : null,
        sentAt: new Date(),
        errorMessage: result.Status[index] !== 'Success' ? result.Error : null
      }));
      
      await this.updateRecipientsStatus(updates);
      
      // Update campaign stats
      await this.updateCampaignStats(campaignId, updates);
      
      // Queue next batch jika masih ada
      const remaining = unsentRecipients.slice(batchSize);
      if (remaining.length > 0) {
        await this.queueNextBatch(campaignId);
      } else {
        await this.completeCampaign(campaignId);
      }
      
      return {
        sent: updates.filter(u => u.status === 'SENT').length,
        failed: updates.filter(u => u.status === 'FAILED').length,
        remaining: remaining.length
      };
      
    } catch (error) {
      console.error('Bulk send error:', error);
      
      // Mark batch as failed and retry later
      await this.handleBatchError(campaignId, batch, error);
      throw error;
    }
  }
  
  // ============ SEND INDIVIDUAL TEMPLATED EMAIL (Untuk testing preview) ============
  async sendTestEmail(to, templateName, templateData) {
    const params = {
      Source: 'PrintFlow <newsletter@printflow.com>',
      Template: templateName,
      Destination: { ToAddresses: [to] },
      TemplateData: JSON.stringify(templateData),
      ConfigurationSetName: 'printflow-bulk-configuration'
    };
    
    return await this.ses.sendTemplatedEmail(params).promise();
  }
  
  // ============ CREATE EMAIL TEMPLATE ============
  async createEmailTemplate(templateData) {
    const params = {
      Template: {
        TemplateName: templateData.name,
        SubjectPart: templateData.subject,
        HtmlPart: templateData.html,
        TextPart: templateData.text
      }
    };
    
    try {
      // Check if template exists
      await this.ses.getTemplate({ TemplateName: templateData.name }).promise();
      // Update existing
      await this.ses.updateTemplate(params).promise();
    } catch (error) {
      // Create new
      await this.ses.createTemplate(params).promise();
    }
    
    return { success: true, templateName: templateData.name };
  }
}

module.exports = new BulkEmailService();
```

```javascript
// backend/src/workers/bulkEmailWorker.js
const { Consumer } = require('sqs-consumer');
const AWS = require('aws-sdk');
const BulkEmailService = require('../services/bulkEmail.service');

class BulkEmailWorker {
  constructor() {
    this.queueUrl = process.env.BULK_EMAIL_QUEUE_URL;
    this.isRunning = false;
    
    this.consumer = Consumer.create({
      queueUrl: this.queueUrl,
      handleMessage: async (message) => {
        const data = JSON.parse(message.Body);
        
        if (data.action === 'START_CAMPAIGN') {
          await BulkEmailService.processBulkCampaign(data.campaignId);
        } else if (data.action === 'NEXT_BATCH') {
          await BulkEmailService.processBulkCampaign(data.campaignId);
        }
      },
      batchSize: 1,
      visibilityTimeout: 300, // 5 minutes
      waitTimeSeconds: 20
    });
    
    this.consumer.on('error', (err) => {
      console.error('Consumer error:', err);
    });
    
    this.consumer.on('processing_error', (err) => {
      console.error('Processing error:', err);
    });
  }
  
  start() {
    if (!this.isRunning) {
      this.consumer.start();
      this.isRunning = true;
      console.log('Bulk email worker started');
    }
  }
  
  stop() {
    if (this.isRunning) {
      this.consumer.stop();
      this.isRunning = false;
      console.log('Bulk email worker stopped');
    }
  }
}
```

```javascript
// backend/src/models/campaign.model.js
const AWS = require('aws-sdk');
const dynamodb = new AWS.DynamoDB.DocumentClient();

const CAMPAIGN_TABLE = 'printflow_campaigns';
const RECIPIENTS_TABLE = 'printflow_campaign_recipients';

// Campaign Schema
// {
//   campaignId: string (PK),
//   name: string,
//   subject: string,
//   templateName: string,
//   status: 'PENDING' | 'SENDING' | 'COMPLETED' | 'CANCELLED' | 'PAUSED',
//   scheduledAt: ISO string,
//   totalRecipients: number,
//   sentCount: number,
//   failedCount: number,
//   openCount: number,
//   clickCount: number,
//   bounceCount: number,
//   complaintCount: number,
//   createdAt: ISO string,
//   completedAt: ISO string (optional)
// }

// Recipient Schema
// {
//   campaignId: string (PK),
//   email: string (SK),
//   status: 'PENDING' | 'SENT' | 'FAILED' | 'BOUNCED' | 'COMPLAINT',
//   name: string,
//   messageId: string,
//   sentAt: ISO string,
//   openedAt: ISO string (optional),
//   clickedAt: ISO string (optional),
//   errorMessage: string (optional)
// }

class CampaignModel {
  async saveCampaign(campaign) {
    const params = {
      TableName: CAMPAIGN_TABLE,
      Item: campaign
    };
    await dynamodb.put(params).promise();
  }
  
  async getCampaign(campaignId) {
    const params = {
      TableName: CAMPAIGN_TABLE,
      Key: { campaignId }
    };
    const result = await dynamodb.get(params).promise();
    return result.Item;
  }
  
  async updateCampaign(campaignId, updates) {
    const updateExpression = [];
    const expressionAttributeValues = {};
    const expressionAttributeNames = {};
    
    Object.keys(updates).forEach((key, index) => {
      const attrKey = `#${key}`;
      const attrValue = `:${key}`;
      updateExpression.push(`${attrKey} = ${attrValue}`);
      expressionAttributeNames[attrKey] = key;
      expressionAttributeValues[attrValue] = updates[key];
    });
    
    const params = {
      TableName: CAMPAIGN_TABLE,
      Key: { campaignId },
      UpdateExpression: `SET ${updateExpression.join(', ')}`,
      ExpressionAttributeNames: expressionAttributeNames,
      ExpressionAttributeValues: expressionAttributeValues
    };
    
    await dynamodb.update(params).promise();
  }
  
  async addRecipients(campaignId, recipients) {
    const items = recipients.map(recipient => ({
      PutRequest: {
        Item: {
          campaignId,
          email: recipient.email,
          name: recipient.name,
          status: 'PENDING',
          createdAt: new Date().toISOString()
        }
      }
    }));
    
    // Batch write (max 25 items per batch)
    for (let i = 0; i < items.length; i += 25) {
      const batch = items.slice(i, i + 25);
      const params = {
        RequestItems: {
          [RECIPIENTS_TABLE]: batch
        }
      };
      await dynamodb.batchWrite(params).promise();
    }
  }
  
  async getUnsentRecipients(campaignId, limit = 50) {
    const params = {
      TableName: RECIPIENTS_TABLE,
      KeyConditionExpression: 'campaignId = :campaignId',
      FilterExpression: 'status = :status',
      ExpressionAttributeValues: {
        ':campaignId': campaignId,
        ':status': 'PENDING'
      },
      Limit: limit
    };
    
    const result = await dynamodb.query(params).promise();
    return result.Items;
  }
  
  async updateRecipientStatus(campaignId, email, status, messageId = null) {
    const params = {
      TableName: RECIPIENTS_TABLE,
      Key: { campaignId, email },
      UpdateExpression: 'SET #status = :status, #sentAt = :sentAt',
      ExpressionAttributeNames: {
        '#status': 'status',
        '#sentAt': 'sentAt'
      },
      ExpressionAttributeValues: {
        ':status': status,
        ':sentAt': new Date().toISOString()
      }
    };
    
    if (messageId) {
      params.UpdateExpression += ', messageId = :messageId';
      params.ExpressionAttributeValues[':messageId'] = messageId;
    }
    
    await dynamodb.update(params).promise();
  }
}

module.exports = new CampaignModel();
```

### B. Lambda untuk Handling Bounce & Complaint

```javascript
// lambda/email-events-handler/index.js
const AWS = require('aws-sdk');
const dynamodb = new AWS.DynamoDB.DocumentClient();
const ses = new AWS.SES();

const SUPPRESSION_TABLE = 'printflow_email_suppression';
const CAMPAIGN_TABLE = 'printflow_campaigns';
const RECIPIENTS_TABLE = 'printflow_campaign_recipients';

// Bounce types & handling strategies
const BOUNCE_ACTIONS = {
  PERMANENT: 'permanent',    // Hard bounce - remove from list
  TRANSIENT: 'transient',     // Soft bounce - retry
  UNDETERMINED: 'undetermined'
};

// Mapping bounce subtype to action
const bounceSubTypeMap = {
  'General': BOUNCE_ACTIONS.PERMANENT,
  'NoEmail': BOUNCE_ACTIONS.PERMANENT,
  'Suppressed': BOUNCE_ACTIONS.PERMANENT,
  'MailboxFull': BOUNCE_ACTIONS.TRANSIENT,
  'MessageTooLarge': BOUNCE_ACTIONS.TRANSIENT,
  'ContentRejected': BOUNCE_ACTIONS.PERMANENT,
  'AttachmentRejected': BOUNCE_ACTIONS.PERMANENT
};

exports.handler = async (event) => {
  console.log('Email event received:', JSON.stringify(event, null, 2));
  
  for (const record of event.Records) {
    const snsMessage = JSON.parse(record.Sns.Message);
    const notificationType = snsMessage.notificationType;
    
    switch (notificationType) {
      case 'Bounce':
        await handleBounce(snsMessage.bounce);
        break;
      case 'Complaint':
        await handleComplaint(snsMessage.complaint);
        break;
      case 'Delivery':
        await handleDelivery(snsMessage.delivery);
        break;
      case 'Open':
        await handleOpen(snsMessage.open);
        break;
      case 'Click':
        await handleClick(snsMessage.click);
        break;
      default:
        console.log('Unknown notification type:', notificationType);
    }
  }
  
  return { statusCode: 200 };
};

// Handle email bounces
async function handleBounce(bounceData) {
  console.log('Processing bounce:', bounceData);
  
  const { bounceType, bounceSubType, bouncedRecipients, timestamp } = bounceData;
  
  for (const recipient of bouncedRecipients) {
    const emailAddress = recipient.emailAddress;
    const action = bounceSubTypeMap[bounceSubType] || BOUNCE_ACTIONS.PERMANENT;
    
    console.log(`Bounce for ${emailAddress}: type=${bounceType}, subType=${bounceSubType}, action=${action}`);
    
    // 1. Add to suppression list
    await addToSuppressionList(emailAddress, {
      reason: 'BOUNCE',
      bounceType,
      bounceSubType,
      timestamp,
      action
    });
    
    // 2. Update campaign recipient status
    await updateRecipientStatus(emailAddress, 'BOUNCED', { bounceType, bounceSubType });
    
    // 3. For permanent bounces, remove from active campaigns
    if (action === BOUNCE_ACTIONS.PERMANENT) {
      await removeFromAllCampaigns(emailAddress);
      
      // Optional: Send notification to admin
      await notifyAdmin('Permanent bounce detected', { email: emailAddress, reason: bounceSubType });
    }
    
    // 4. For transient bounces, schedule retry
    if (action === BOUNCE_ACTIONS.TRANSIENT) {
      await scheduleRetry(emailAddress, bounceData);
    }
  }
}

// Handle complaints (user marked as spam)
async function handleComplaint(complaintData) {
  console.log('Processing complaint:', complaintData);
  
  const { complainedRecipients, timestamp, complaintFeedbackType } = complaintData;
  
  for (const recipient of complainedRecipients) {
    const emailAddress = recipient.emailAddress;
    
    console.log(`Complaint from ${emailAddress}: ${complaintFeedbackType}`);
    
    // 1. Add to suppression list (CRITICAL - spam complaint)
    await addToSuppressionList(emailAddress, {
      reason: 'COMPLAINT',
      complaintType: complaintFeedbackType,
      timestamp,
      action: BOUNCE_ACTIONS.PERMANENT
    });
    
    // 2. Immediately remove from all campaigns
    await removeFromAllCampaigns(emailAddress);
    
    // 3. Update user preferences (mark as unsubscribed)
    await updateUserSubscription(emailAddress, false);
    
    // 4. Send critical alert to admin
    await notifyAdmin('⚠️ SPAM COMPLAINT - ACTION REQUIRED', {
      email: emailAddress,
      complaintType: complaintFeedbackType,
      timestamp
    });
  }
}

// Handle successful deliveries
async function handleDelivery(deliveryData) {
  console.log('Processing delivery:', deliveryData);
  
  const { recipients, timestamp, processingTimeMillis } = deliveryData;
  
  for (const recipient of recipients) {
    const emailAddress = recipient;
    
    // Update delivery status
    await updateRecipientStatus(emailAddress, 'DELIVERED', {
      deliveredAt: timestamp,
      processingTime: processingTimeMillis
    });
    
    // Update campaign delivery stats
    await updateCampaignStats(emailAddress, 'delivered');
  }
}

// Handle email opens
async function handleOpen(openData) {
  console.log('Processing open:', openData);
  
  const { ipAddress, timestamp, userAgent } = openData;
  
  // Extract campaign and recipient from open data
  const campaignId = openData.tags?.campaignId;
  const emailAddress = openData.destination;
  
  if (campaignId && emailAddress) {
    await updateRecipientOpen(emailAddress, campaignId, {
      openedAt: timestamp,
      ipAddress,
      userAgent
    });
    
    await updateCampaignStats(campaignId, 'open');
  }
}

// Handle email clicks
async function handleClick(clickData) {
  console.log('Processing click:', clickData);
  
  const { link, timestamp, ipAddress, userAgent } = clickData;
  
  const campaignId = clickData.tags?.campaignId;
  const emailAddress = clickData.destination;
  
  if (campaignId && emailAddress) {
    await updateRecipientClick(emailAddress, campaignId, {
      clickedAt: timestamp,
      link,
      ipAddress,
      userAgent
    });
    
    await updateCampaignStats(campaignId, 'click');
  }
}

// Helper: Add email to suppression list
async function addToSuppressionList(email, data) {
  const params = {
    TableName: SUPPRESSION_TABLE,
    Item: {
      email,
      reason: data.reason,
      details: data,
      createdAt: new Date().toISOString(),
      expiresAt: data.action === BOUNCE_ACTIONS.PERMANENT 
        ? null  // Permanent - never expire
        : new Date(Date.now() + 7 * 24 * 60 * 60 * 1000).toISOString() // Transient - 7 days
    }
  };
  
  await dynamodb.put(params).promise();
  
  // Also add to SES global suppression list
  if (data.reason === 'COMPLAINT' || data.action === BOUNCE_ACTIONS.PERMANENT) {
    await ses.putSuppressedDestination({
      EmailAddress: email,
      Reason: data.reason === 'COMPLAINT' ? 'COMPLAINT' : 'BOUNCE'
    }).promise().catch(e => console.error('SES suppression error:', e));
  }
}

// Helper: Remove email from all campaigns
async function removeFromAllCampaigns(email) {
  // Get all campaigns where email is pending
  const params = {
    TableName: RECIPIENTS_TABLE,
    IndexName: 'email-index',
    KeyConditionExpression: 'email = :email',
    FilterExpression: 'status = :status',
    ExpressionAttributeValues: {
      ':email': email,
      ':status': 'PENDING'
    }
  };
  
  const result = await dynamodb.query(params).promise();
  
  for (const item of result.Items) {
    await dynamodb.update({
      TableName: RECIPIENTS_TABLE,
      Key: {
        campaignId: item.campaignId,
        email: item.email
      },
      UpdateExpression: 'SET #status = :status, #removedAt = :removedAt',
      ExpressionAttributeNames: {
        '#status': 'status',
        '#removedAt': 'removedAt'
      },
      ExpressionAttributeValues: {
        ':status': 'REMOVED',
        ':removedAt': new Date().toISOString()
      }
    }).promise();
  }
}

// Helper: Update user subscription preference
async function updateUserSubscription(email, isSubscribed) {
  // Update in User database
  const params = {
    TableName: 'printflow_users',
    Key: { email },
    UpdateExpression: 'SET #subscribed = :subscribed, #updatedAt = :updatedAt',
    ExpressionAttributeNames: {
      '#subscribed': 'newsletter_subscribed',
      '#updatedAt': 'updatedAt'
    },
    ExpressionAttributeValues: {
      ':subscribed': isSubscribed,
      ':updatedAt': new Date().toISOString()
    }
  };
  
  await dynamodb.update(params).promise();
}

// Helper: Schedule retry for transient bounces
async function scheduleRetry(email, bounceData) {
  // Add to retry queue with exponential backoff
  const retryCount = await getRetryCount(email);
  const delays = [1, 5, 15, 30, 60]; // minutes
  
  if (retryCount < delays.length) {
    const delayMinutes = delays[retryCount];
    const retryAt = new Date(Date.now() + delayMinutes * 60 * 1000);
    
    await addToRetryQueue(email, retryAt, bounceData);
    console.log(`Scheduled retry for ${email} in ${delayMinutes} minutes (attempt ${retryCount + 1})`);
  } else {
    console.log(`Max retries exceeded for ${email}, moving to suppression list`);
    await addToSuppressionList(email, {
      reason: 'MAX_RETRIES_EXCEEDED',
      bounceData,
      action: BOUNCE_ACTIONS.PERMANENT
    });
  }
}

// Helper: Send notification to admin
async function notifyAdmin(subject, data) {
  // Send to SNS for admin notification
  const sns = new AWS.SNS();
  await sns.publish({
    TopicArn: process.env.ADMIN_ALERT_TOPIC,
    Subject: `[PrintFlow Email] ${subject}`,
    Message: JSON.stringify(data, null, 2)
  }).promise();
}
```

```jsx
          className="w-full p-3 border rounded-lg"
          onChange={(e) => {
            setSelectedCampaign(e.target.value);
            fetchAnalytics(e.target.value);
          }}
          value={selectedCampaign || ''}
        >
          <option value="">Select Campaign</option>
          {campaigns.map(campaign => (
            <option key={campaign.campaignId} value={campaign.campaignId}>
              {campaign.name} - {new Date(campaign.createdAt).toLocaleDateString()}
            </option>
          ))}
        </select>
      </div>
      
      {analytics && (
        <>
          {/* Summary Cards */}
          <div className="grid grid-cols-1 md:grid-cols-4 gap-4 mb-6">
            <div className="bg-white rounded-lg shadow p-4">
              <p className="text-gray-500 text-sm">Sent</p>
              <p className="text-2xl font-bold">{analytics.totalSent?.toLocaleString()}</p>
            </div>
            <div className="bg-white rounded-lg shadow p-4">
              <p className="text-gray-500 text-sm">Open Rate</p>
              <p className="text-2xl font-bold">{analytics.openRate}%</p>
            </div>
            <div className="bg-white rounded-lg shadow p-4">
              <p className="text-gray-500 text-sm">Click Rate</p>
              <p className="text-2xl font-bold">{analytics.clickRate}%</p>
            </div>
            <div className="bg-white rounded-lg shadow p-4">
              <p className="text-gray-500 text-sm">Bounce Rate</p>
              <p className="text-2xl font-bold text-red-600">{analytics.bounceRate}%</p>
            </div>
          </div>
          
          {/* Engagement Timeline */}
          <div className="bg-white rounded-lg shadow p-6 mb-6">
            <h2 className="text-lg font-semibold mb-4">Engagement Timeline</h2>
            <ResponsiveContainer width="100%" height={300}>
              <LineChart data={engagementData}>
                <XAxis dataKey="date" />
                <YAxis />
                <Tooltip />
                <Legend />
                <Line type="monotone" dataKey="sent" stroke="#8884d8" />
                <Line type="monotone" dataKey="opened" stroke="#82ca9d" />
                <Line type="monotone" dataKey="clicked" stroke="#ffc658" />
              </LineChart>
            </ResponsiveContainer>
          </div>
          
          {/* Bounce Analysis */}
          <div className="grid grid-cols-1 md:grid-cols-2 gap-6">
            <div className="bg-white rounded-lg shadow p-6">
              <h2 className="text-lg font-semibold mb-4">Bounce Analysis</h2>
              <ResponsiveContainer width="100%" height={250}>
                <BarChart data={bounceRates}>
                  <XAxis dataKey="name" />
                  <YAxis />
                  <Tooltip />
                  <Bar dataKey="value" fill="#dc3545" />
                </BarChart>
              </ResponsiveContainer>
            </div>
            
            {/* Top Performing Content */}
            <div className="bg-white rounded-lg shadow p-6">
              <h2 className="text-lg font-semibold mb-4">Top Performing Links</h2>
              <div className="space-y-3">
                {analytics.topLinks?.map((link, i) => (
                  <div key={i} className="flex justify-between items-center p-3 bg-gray-50 rounded">
                    <span className="text-sm truncate flex-1">{link.url}</span>
                    <span className="font-semibold">{link.clicks} clicks</span>
                  </div>
                ))}
              </div>
            </div>
          </div>
          
          {/* Suppression List */}
          <div className="bg-white rounded-lg shadow p-6 mt-6">
            <h2 className="text-lg font-semibold mb-4">Suppression List</h2>
            <table className="w-full">
              <thead className="bg-gray-50">
                <tr>
                  <th className="p-3 text-left">Email</th>
                  <th className="p-3 text-left">Reason</th>
                  <th className="p-3 text-left">Date</th>
                  <th className="p-3 text-left">Action</th>
                </tr>
              </thead>
              <tbody>
                {analytics.suppressedEmails?.map((item, i) => (
                  <tr key={i} className="border-t">
                    <td className="p-3">{item.email}</td>
                    <td className="p-3">
                      <span className={`px-2 py-1 rounded text-xs ${
                        item.reason === 'COMPLAINT' ? 'bg-red-100 text-red-800' :
                        item.reason === 'BOUNCE' ? 'bg-yellow-100 text-yellow-800' :
                        'bg-gray-100 text-gray-800'
                      }`}>
                        {item.reason}
                      </span>
                    </td>
                    <td className="p-3">{new Date(item.createdAt).toLocaleDateString()}</td>
                    <td className="p-3">
                      <button 
                        onClick={() => removeFromSuppression(item.email)}
                        className="text-blue-600 hover:underline text-sm"
                      >
                        Remove
                      </button>
                    </td>
                  </tr>
                ))}
              </tbody>
            </table>
          </div>
        </>
      )}
    </div>
  );
};

export default EmailAnalytics;
```
